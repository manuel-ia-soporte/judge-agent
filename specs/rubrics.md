# Rubric Definitions

## Scoring Scale
- 0 = Fail
- 1 = Partial
- 2 = Pass

## Rubric Dimensions
- Factual Accuracy
- Source Fidelity  (SEC / EDGAR grounding)
- Regulatory Compliance
- Financial Reasoning Quality
- Materiality & Relevance
- Completeness
- Consistency
- Temporal Validity
- Risk Awareness
- Clarity & Interpretability
- Uncertainty Handling
- Actionability (optional)

Each rubric must:
- Be independently evaluable
- Produce an explanation
- Reference evidence or rules
